## Labs Completed

### 1. Static Website Deployment
**Services Used:** Amazon S3, CloudFront
- Deployed a static website using Amazon S3 for storage
- Configured CloudFront as a content delivery network for fast global access
- Applied IAM bucket policies to control public access securely

### 2. Identity and Access Management (IAM)
**Services Used:** AWS IAM
- Configured users, groups, roles, and policies
- Implemented least-privilege access control across AWS resources
- Demonstrated AWS security best practices

### 3. Amazon RDS Database Deployment
**Services Used:** Amazon RDS
- Provisioned and configured a relational database instance
- Configured security groups and tested connectivity
- Validated database performance and access controls

### 4. Data Encryption at Rest
**Services Used:** AWS KMS, AWS Encryption
- Implemented encryption at rest using AWS-native encryption tools
- Configured AWS Key Management Service (KMS) policies
- Applied security controls to protect sensitive cloud-stored data

### 5. Serverless Architecture Implementation
**Services Used:** AWS Lambda, API Gateway, S3
- Designed and deployed a serverless application architecture
- Built scalable cloud solutions without managing infrastructure
- Demonstrated cost-efficient cloud design principles

### 6. Application Security with Amazon Cognito
**Services Used:** Amazon Cognito, AWS IAM
- Secured a cloud application using Amazon Cognito
- Configured user pools and identity pools
- Managed authentication and authorization for application users

### 7. Amazon Elastic File System (EFS)
**Services Used:** Amazon EFS, EC2
- Provisioned and mounted an EFS file system to EC2 instances
- Enabled shared scalable cloud storage across multiple compute resources
